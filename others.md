## Glossary

- **Agent** — an LLM operating in a loop with access to tools (file I/O, shell, web, etc.), able to take multiple steps toward a goal.
- **Agentic loop** — the cycle of the agent observing state, deciding an action, executing it, and observing the result again.
- **MCP (Model Context Protocol)** — an open protocol for connecting AI applications to external tools and data sources.
- **Context window** — the amount of text (measured in tokens) a model can process at once; managing it well affects agent quality on large codebases.
- **Tool use / function calling** — a model's ability to invoke external functions/tools as part of generating a response.
